# Vendeta 🗺️

> Dezentrales Preis-Transparenz-Netzwerk für den Alltag.
> Menschen melden Schnäppchen und Wucherpreise in Echtzeit —
> verankert auf Blockchain, belohnt mit IFR-Token-Credits.

## Vision
Eine offene, community-getriebene Plattform wo Konsumenten
Preise für Waren und Dienstleistungen in ihrer Umgebung
transparent machen — und dafür Rewards erhalten.

## Tech Stack
| Schicht | Technologie |
|---|---|
| Core Library | Rust |
| Backend API | Rust + Axum |
| Mobile App | Flutter (Android + iOS) |
| Smart Contracts | Solidity (Base L2) |
| Datenbank | PostgreSQL + PostGIS |
| Maps | MapLibre + OpenStreetMap |
| Token / Rewards | IFR (ifrunit.tech) |
| Blockchain | Ethereum Mainnet + Base L2 |
